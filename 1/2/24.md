# 开发/发布证书管理

```
常规的证书创建流程和管理, 互相传p12文件是不是相当的麻烦？
我们用AFastlane托管这一切
```

---

### 基础环境

```zsh
# 检查fastlane环境
fastlane env
```
### 配置

```
项目文件下fastlane文件即为配置,
请不要随意更改,
```

具体参考[fastlane文档](https://docs.fastlane.tools/)

### 管理证书

```zsh
# 刷新本地证书, 如果没有权限请询问管理员
fastlane refresh
```
