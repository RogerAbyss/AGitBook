# ruby/gem运行环境

---

### 基础环境

```zsh
# MacOS已经集成了基础环境, homebrew
ruby -v
gem -v
```

### bundler安装

```zsh
sudo gem install bundler
```

### 通过bundler安装Gemfile

项目文件下有Gemfile,
```zsh
# 可以安装我们需要的依赖
# email 邮件发送
# fastlane 自动化运维
# fir-cli 上传fir.im
# cocoapods 三方库管理
# jazzy 文档生成
bundle install
```
