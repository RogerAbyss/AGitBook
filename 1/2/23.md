# node/npm运行环境

---

### 基础环境

```zsh
# MacOS已经集成了基础环境, homebrew
node -v
npm -v
```

### 通过npm安装packages.json

项目文件下有packages.json
```zsh
# 可以安装我们需要的依赖
# commitizen 提交格式规范
# husky git钩子
# validate-commit-msg 本地提交格式检查
# cz-conventional-changelog 生成CHANGELOG
npm install
```

### npm功能

- [x] 已经git hook, git请通过``git cz``提交
- [x] npm run changelog : 生成CHANGELOG.md
