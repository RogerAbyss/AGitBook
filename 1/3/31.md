# Git分支管理

---

| **分支** | **属性** | **作用** |
| :--- | :--- | | :--- |
| **master** | 主要 | 主要存档分支 |
| **Develop** | 主要 | 主要开发分支  |
| **AppStore** | 主要 | 主要Appstore版本分支 |
| 其他Feature/Fix | 次要 | **每次新功能,修改bug请新建分支** |

**master,Develop,AppStore是保护分支, 不能被push -f**

```
为了分支与生成的提交记录的美观, 
请严格按照规范书写每一个git commit
并且尽量在私有分支备份的情况下使用git rebase优化提交记录。
```

具体参考下文[Git提交规范与CHANGELOG](/1/3/32.md)
